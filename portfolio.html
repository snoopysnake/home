GITHUB

Camper Creator (2018)

A sprite/avatar creator to promote Bitcamp, a student-run hackathon at the University of Maryland. Written in JavaScript. Most artwork drawn by me (with help from Andy and Elli).


CMSC320 Final (2017)

My final project for CMSC320 (Introduction to Data Science), where I attempt to find the relationship between high school football player rating vs. football win percentage at the University of Maryland. Written in Python.


JTAF UI (2017)

UI for the test automation framework (JTAF) used at FINRA. I attempted to create an easier way to write automated tests. Written in Java.


FormCheck (2016)

Microsoft Kinect application used to check lifting form, mainly squat depth. Created for Bitcamp 2016. Major credit goes to David. Written in C#.


Poltergeist (2015)

Simple Unity game created for MHacks 2015 using EA's Pathfinder API. Written in C# and JavaScript.


PDF417 Encrypt (2015)

Web app created for VT Hacks 2015 that encodes/decodes both text and images into a PDF417 barcode. Written in JavaScript.


Sustainability Simulator 2014 (2014)

Flash game that educates sustainability by demonstrating how to recycle and compost. Created for Daemon Dash 2014 with Andrew, Bohr, and Dan. All art drawn by me. Written in ActionScript.
